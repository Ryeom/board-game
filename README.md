
# 🎲 Board Game Backend

LLM 기반 가상 플레이어와 함께 즐기는 온라인 보드게임 플랫폼. 
현재는 **하나비**를 중심으로 멀티플레이 게임 기능을 구현하고 있으며, 확장 가능한 구조로 다양한 게임 모드를 지원할 예정입니다.

## 🏗️ 기술 스택

- **Go** (백엔드)
- **WebSocket** (실시간 통신)
- **Redis** (세션 관리 예정)
- **Docker** (개발 및 배포 환경 구성 예정)
- **LLM API 연동** (가상 플레이어)

### 🎮 게임 방 기능

- [x] 방 생성하기
